# BipNews - Сайт новостей техникума "Бизнес и Право"

## Описание
Полноценная система управления новостями для техникума "Бизнес и Право".

### Возможности:
- ✅ Регистрация с проверкой возраста (14+)
- ✅ Авторизация и выход
- ✅ Создание, редактирование и удаление новостей (администраторы)
- ✅ Комментарии к новостям
- ✅ Лайки и дизлайки для комментариев
- ✅ Профили пользователей с аватарами
- ✅ Личные страницы пользователей с историей комментариев
- ✅ Админ-панель для управления контентом и пользователями
- ✅ Счётчик просмотров новостей
- ✅ Разграничение ролей: пользователь, модератор, администратор

## Технологии
- Backend: чистый PHP
- Frontend: HTML, CSS, JavaScript
- База данных: MySQL

## Установка

### Требования
- PHP 7.4 или выше
- MySQL 5.7 или выше
- Веб-сервер Apache/Nginx (рекомендуется Apache)
- Расширение PHP PDO MySQL

### Шаги установки

1. **Скопируйте файлы проекта** в директорию веб-сервера:
   ```
   e:\programs\xampp\htdocs\work\bipNews
   ```

2. **Настройте базу данных** в файле `config/database.php`:
   ```php
   define('DB_HOST', 'localhost');
   define('DB_NAME', 'bipnews');
   define('DB_USER', 'root');
   define('DB_PASS', '');
   ```

3. **Запустите установку**:
   Откройте в браузере: `http://localhost/bipNews/install.php`
   
   Скрипт автоматически создаст базу данных и все необходимые таблицы.

4. **Удалите файл `install.php`** после установки (по соображениям безопасности).

5. **Войдите в админ-панель**:
   - URL: `http://localhost/bipNews/admin/index.php`
   - Логин: `admin`
   - Пароль: `admin123`
   
   ⚠️ **Обязательно смените пароль после первого входа!**

## Роли пользователей

### Пользователь (user)
- Просмотр новостей
- Оставление комментариев
- Голосование за комментарии (лайки/дизлайки)
- Редактирование своего профиля

### Модератор (moderator)
- Все права пользователя
- Удаление любых комментариев

### Администратор (admin)
- Все права модератора
- Создание новостей
- Редактирование новостей
- Удаление новостей
- Управление пользователями (изменение ролей, удаление)
- Доступ к админ-панели
